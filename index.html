Distance Detecting Alarm

The Distance Detecting Alarm System is a sensor-based project designed to detect objects within a specific range and trigger an alert when the distance threshold is reached. The system demonstrates the use of ultrasonic sensors, real-time monitoring, and automated responses. This project helped me understand how sensor data can be processed and used for safety and security applications.

Studymate AI

The AI Study Assistant is a tool designed to help students study more efficiently by analyzing lecture slides and automatically generating summaries and quiz questions based on the content. By extracting key information from learning materials, the system helps students quickly review important concepts and test their understanding.

VR Smart Farming

The project combines IoT technology with virtual reality to create an immersive plant monitoring system. Using ESP32 sensors, environmental data such as temperature and humidity are collected and displayed inside a VR interface. Users can observe plant conditions and control the water pump within a virtual farming environment, making the monitoring process more interactive and intuitive.
